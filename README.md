## ABOUT PROJECT

This is a clone of Apple's iPhone 15 Pro website using React.js and TailwindCSS. 

## FEATURES

- **Subtle Smooth Animations using GSAP**

- **3D Model Rendering with Different Colors and Sizes**

- **Custom Video Carousel (made with GSAP)**

- **Completely Responsive**
